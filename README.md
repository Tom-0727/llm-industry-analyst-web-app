# LLM Industry Analyst Web App

AI-powered investment research dashboard focused on Chinese LLM companies.

## Tracked Stocks

| Stock | Ticker | Description |
|-------|--------|-------------|
| MiniMax Group | 0100.HK | AI model company (Hailuo AI, Talkie) |
| Zhipu AI | 2513.HK | AGI pioneer (GLM-5, ChatGLM) |

## Features

- Real-time stock price tracking and charts
- AI-generated predictions with confidence scores
- Prediction tracking and accuracy measurement
- Company profiles and key metrics

## Data

- `data/minimax_prices.json` - MiniMax daily OHLCV
- `data/zhipu_prices.json` - Zhipu daily OHLCV
- `data/predictions.json` - Prediction records
